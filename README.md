# PawTrack

"PawTrack" es una herramienta de gestión para clínicas veterinarias que permite un seguimiento completo de los pacientes animales, desde el registro de síntomas hasta las fechas de ingreso y alta.

Tecnologías usadas:

- React
- TypeScript
- React Hook Form
- Zustand
- Tailwind
